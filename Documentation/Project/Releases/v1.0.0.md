# Release v1.0.0

## Why this version matters
v1.0.0 is the first stable baseline for running benchmarks reliably on ST Edge AI Developer Cloud with persistent result tracking.

## Highlights
- SQLite-first storage with CSV export compatibility.
- Benchmark traceability with core_command, run_id, benchmark_id.
- Reference tagging and version history queries.
- Safer metric parsing and actionable error hints.

## How to use
- Run interactive menu: python main.py
- Run full regression: python run_ci_tests.py

## Known limitations
- Some advanced memory options remain board-dependent and can be rejected in NPU mode.

## Support status
- Legacy stable baseline kept available through Git tag and GitHub Release.
