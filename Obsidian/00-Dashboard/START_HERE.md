1. Follow plugin setup: [PLUGIN_SETUP.md](PLUGIN_SETUP.md)
2. Create your first requirement from template: [../03-Templates/REQUIREMENT_TEMPLATE.md](../03-Templates/REQUIREMENT_TEMPLATE.md)
3. Open backlog dashboard: [BACKLOG.md](BACKLOG.md)
4. Start the Codex routine from your requirement note.
